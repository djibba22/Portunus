Store My Key

Safely store your API Key off of Github

We have all seen the warning with storing sensitive data on github. When you are using free api keys and deploying to gh-pages this can be a pain. Without server side code there is no place to safely store these credentials, yet make them available on gh-pages.

This is where Portunus steps in. A simple solution to a painful problem. Create a new name for your key and store the actual value some where else. Simply make a request from your application to load your key and github will never know a thing.
